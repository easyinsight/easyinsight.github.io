APIs for Developers

Want to publish custom data into Easy Insight? You can use our APIs to publish any custom data into the system for reporting and analytics.

Introduction

The Easy Insight API provides developers with the capability of publishing any custom data sets into the system.

Types of APIs

Easy Insight provides two distinct API variants, SOAP and REST. The two APIs provide the same set of functionality--which you use just depends on your particular development preferences.

Benefits

Using a system that Easy Insight doesn't connect to yet? Use the Easy Insight API to publish in data from that system and you can start building the same reports and dashboards against that data as you can against any of the default connection types.
