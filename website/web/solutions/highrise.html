Installation Instructions
First, if not already enabled, you need to be enable API access of your Highrise account. Once that's done, click the button to the left to install the Highrise solution into your account. After installing the data source, you'll be prompted for your Highrise URL and credentials. Once you hit Create with a valid URL and credentials, Easy Insight will pull over key aggregate data from your Highrise installation through the API. This process may take a little time, depending on the size of your installation.
