Unchecked API

addRow

Params:

  • dataSourceName - string - the name of the data source to be added to
  • row - row - the row to be added

The addRow method on the unchecked API adds a row to the data source named in the dataSourceName field. If there is no data source with that name, the data source will be created. If there are multiple data sources with that name, then the row is added to the first data source, so when using the unchecked API, use care in choosing data source names.

See Also:

row

Example

            <addRow>
                <dataSourceName>User Counts</dataSourceName>
                <row>
                    <numberPairs>
                        <numberPair>
                            <key>Count</key>
                            <value>76</value>
                        </numberPair>
                    </numberPairs>
                    <stringPairs>
                        <stringPair>
                            <key>Type</key>
                            <value>Free</value>
                        </stringPair>
                    </stringPairs>
                </row>
            </addRow>
